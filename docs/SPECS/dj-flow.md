# DJ Flow Spec

## Core DJ tasks
- authenticate
- manage playlists
- manage songs
- review guest requests
- move selected requests into active queue
- reorder queue
- mark now playing
- configure guest-visible queue settings
- manage multiple song lists (create/select/edit/delete) and list-scoped song import/add/remove

## Important UI rules
- CRUD actions should be simple and obvious
- bulk complexity is not required in first pass
- page states must include empty/loading/error/success
