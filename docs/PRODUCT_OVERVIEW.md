# Product Overview

## Product
QR-based song request frontend for guests and DJs.

## Repo role
This repository delivers the frontend presentation/demo layer first, using mock contracts and mock HTTP responses that mirror the future Laravel REST API.

## Primary frontend surfaces
- Public guest QR page
- DJ dashboard
- Queue and now-playing management
- Basic auth screens for demo completeness
- Settings screens for queue visibility

## Design direction
- Mobile-first
- Modern but not overdesigned
- Clean, polished, slightly glossy/plastic visual direction
- Lightweight and easy to extend
- CRUD-friendly IA and component structure
